Chapter 2: Features

StPageFlip offers many great features:

  • Realistic page turning animations
  • Mobile device support
  • Portrait and landscape modes
  • Soft and hard page types
  • No dependencies
Feature Diagram
2

Chapter 3: Usage

Using StPageFlip is simple. Just create your HTML structure with pages, then initialize the library:

const pageFlip = new St.PageFlip(
  document.getElementById('book'),
  { width: 400, height: 600 }
);

Then load your pages and enjoy the smooth animations!

3

Chapter 4: Configuration

You can customize many aspects of the book behavior:

Settings Panel
  • width/height: Page dimensions
  • showCover: Single page mode for covers
  • flippingTime: Animation duration
  • drawShadow: Enable/disable shadows
4

Chapter 5: Events

The library provides several useful events:

pageFlip.on('flip', (e) => {
  console.log('Page:', e.data);
});
Event Flow

Events include: flip, changeOrientation, changeState, init, and update.
